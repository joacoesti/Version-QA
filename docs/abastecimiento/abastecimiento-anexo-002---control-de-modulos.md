# 1\. Objetivo del anexo

Concentrar los controles por producto, familia y módulo que Abastecimiento debe mirar durante el relevamiento y la reposición\. Este anexo permite consultar qué revisar, qué capacidad tomar como referencia y qué hacer ante desvíos sin sobrecargar el documento principal de tareas diarias\.

# 2\. Regla operativa

__Regla de control__

Antes de cruzar o reponer, el producto debe estar en condición vendible: ubicación correcta, etiqueta visible, vencimiento controlado, presentación adecuada y criterio FIFO aplicado\.

# 4\. Criterios transversales

__Criterio__

__Aplicación práctica__

FIFO

Lo que vence primero debe quedar adelante o arriba\. No tapar producto próximo a vencer con producto nuevo\.

Etiqueta

Debe estar visible, legible y orientada para lectura rápida\. Producto sin etiqueta no se exhibe\.

Presentación

No llenar de cualquier forma: la exhibición debe quedar prolija, ordenada y vendible\.

Calidad visual

Producto con color, envase, vacío o estado dudoso se separa y se consulta antes de exhibir\.

Comunicación

Faltante detectado debe comunicarse; si no se avisa, la góndola puede quedar vacía aunque Cocina tenga capacidad de respuesta\.
