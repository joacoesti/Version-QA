# 1\. Objetivo del anexo

Estandarizar el uso de las planillas de reposición y stock interno para que Abastecimiento pueda contar, registrar, reponer y comunicar faltantes sin perder datos. El objetivo es que el control no dependa de memoria ni de comentarios verbales aislados.

# 2\. Regla operativa

**Regla de registro**

Primero se cuenta, después se repone. Si se retira producto del freezer o cámara interna y no se anota, el stock real se pierde.

# 3\. Planilla de reposición de congelados

1\. Contar cantidades de cada producto en góndola.

2\. Empanadas congeladas: anotar paquetes faltantes para llegar a 12 por módulo.

3\. Tartas congeladas: anotar unidades faltantes para llegar a 20.

4\. Ravioles congelados: anotar unidades faltantes para llegar a 15.

5\. Comidas al vacío: anotar unidades faltantes para llegar a 15.

6\. Completar la planilla indicando la cantidad que hay que reponer.

# 4\. Planilla de reposición de isla

1\. Contar cantidades de cada producto en góndola.

2\. Ensaladas: anotar unidades faltantes para llegar a 60.

3\. Salsa de tomate: anotar unidades faltantes para llegar a 4.

4\. Salsa blanca: anotar unidades faltantes para llegar a 2.

# 5\. Planilla de stock interno

1\. Contar productos disponibles en cada freezer o sector de cámara interna.

2\. Respetar ubicación fija: cada producto debe estar en el freezer rotulado.

3\. Registrar el total en la planilla.

4\. No incluir lo que ya está exhibido en salón.

5\. Avisar al Jefe de Cocina cuando el stock real esté por debajo del mínimo definido.

# 7\. Errores que debe evitar el rol

**Error frecuente**

**Impacto**

**Acción preventiva**

Reponer sin contar

Se pierde el dato real de faltantes.

Completar planilla antes de cruzar.

Contar stock exhibido como stock interno

Se sobreestima disponibilidad.

Separar stock de salón vs. stock de cámara/freezer.

Retirar producto sin anotar

El stock real queda desactualizado.

Registrar cada retiro o reposición.

No comunicar quiebres

Cocina no recibe alerta a tiempo.

Avisar apenas se detecta faltante crítico.

No respetar ubicación fija

Se pierde tiempo buscando y se rompe FIFO.

Guardar cada producto en freezer rotulado.
